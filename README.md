# 💖 Encontro Elas Com Ele

Sistema web desenvolvido para gerenciamento de encontros femininos, permitindo cadastro, login e visualização de mesas (eventos).

---

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de facilitar a organização de eventos recorrentes, onde participantes podem se cadastrar e visualizar próximas mesas de encontro.

---

## 🚀 Funcionalidades

- ✅ Cadastro de usuárias  
- ✅ Login  
- ✅ Visualização de mesas (modal popup)  
- ✅ Interface moderna com animações  
- ✅ Layout responsivo e estilizado  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Node.js  
- Express  
- MySQL
